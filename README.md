# 工作提示词合集 (Work Prompts Collection)

本项目是一个汇集了 300 个高质量、结构化 AI 提示词（Prompts）的集合。旨在帮助开发者、知识工作者和创作者更高效地使用如 Claude 等 AI 助手，全面提升工作流效率。

## 📁 目录结构

所有提示词分类存放在 `prompts/` 目录下，共分为 6 大核心类别：

- **[类别 1：编码与调试](./prompts/category-1.md) (提示词 1-50)**
  *高级工程师视角的代码审查、调试、重构、单元测试、性能优化以及语言转换等。*
- **[类别 2：AI 工作流](./prompts/category-2.md) (提示词 51-100)**
  *针对通用 AI 对话模型设定的工作流引导词。*
- **[类别 3：研究与分析](./prompts/category-3.md) (提示词 101-150)**
  *用于数据分析、文档提炼、市场调研及复杂信息整理。*
- **[类别 4：自动化](./prompts/category-4.md) (提示词 151-200)**
  *工作流自动化脚本编写与日常繁杂任务自动化处理提示词。*
- **[类别 5：内容创作](./prompts/category-5.md) (提示词 201-250)**
  *辅助文案撰写、文档生成、文章润色与创意工作等。*
- **[类别 6：生产力系统](./prompts/category-6.md) (提示词 251-300)**
  *包含任务拆解规划、时间管理建议、目标设定等提升个人和团队生产力的方法。*

## 🚀 如何使用

1. 进入 `prompts/` 文件夹。
2. 根据你的需求找到对应分类的 Markdown 文件。
3. 复制相应的提示词模板。
4. 替换其中的 `[占位符]`（如 `[粘贴代码]`、`[错误信息]` 等）为你实际的内容。
5. 将组装好的内容发送给 AI 助手即可获得高质量答复。
